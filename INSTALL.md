# Installation Guide

## Quick Installation

### Step 1: Download
- Go to [Releases](https://github.com/MattGHarvey/MassLinkOpener/releases)
- Download the latest `mass-link-opener-v*.zip` file
- Extract the ZIP file to a folder on your computer

### Step 2: Install in Browser
1. Open your browser (Chrome or Edge)
2. Navigate to the extensions page:
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
3. Enable **Developer mode** (toggle in top-right corner)
4. Click **"Load unpacked"**
5. Select the folder where you extracted the extension
6. The extension will appear in your browser toolbar

### Step 3: Start Using
1. Visit any webpage with links
2. Click the Mass Link Opener icon
3. Enter a search phrase
4. Find and open matching links!

## Troubleshooting

**Extension not working?**
- Make sure Developer mode is enabled
- Try refreshing the webpage
- Check that you're not on a browser internal page (chrome://, edge://)

**Can't find the extension icon?**
- Click the puzzle piece icon in your toolbar
- Find "Mass Link Opener" and pin it

**No links found?**
- Try a broader search term
- Disable case-sensitive search
- Make sure the page has finished loading

## Updating

When a new version is released:
1. Download the new ZIP file
2. Remove the old extension from chrome://extensions/
3. Install the new version following the steps above
